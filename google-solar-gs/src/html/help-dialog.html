Solar System Designer — Help

Getting Started

  1. Go to Solar API > Settings to enter your Google Maps Platform API key.
  2. Open the sidebar via Solar API > Open Sidebar.
  3. In the Location tab, enter an address or click on the map to select a location.
  4. Click Fetch Solar Data to retrieve building insights and data layers.
  5. Explore the tabs: Building, Layers, Charts, and Cost to analyze the solar potential.
  6. Use Inject to Sheet buttons to export data into spreadsheet tabs.

Tabs Overview

  • Location — Select location, toggle satellite/street view, fetch solar data.
  • Building — View building summary, roof segments, individual panel layout, and system configuration.
  • Layers — Visualize GeoTIFF data layers: annual flux, monthly flux, hourly shade, and DSM/RGB imagery.
  • Charts — Interactive charts of energy metrics, panel efficiency, and sunshine distribution.
  • Cost — Financial analysis: installation costs, utility savings, and break-even estimates.

Key Data Points

  • Annual Flux — Solar energy received per square meter per year (kWh/m²/yr).
  • Yearly Energy DC — Total DC power the panels can generate in a year (kWh).
  • Panel Capacity — Rated power output of each solar panel (Watts).
  • DC-to-AC Derate — Conversion efficiency from DC to AC power (typically ~0.85).
  • Efficiency Depreciation — Annual panel performance degradation factor.
  • Sunshine Quantiles — Distribution of solar irradiance across the roof surface.
API Costs: The Google Solar API has usage-based pricing. Check Google's billing docs for current rates. Building Insights requests and Data Layers requests are billed separately.
